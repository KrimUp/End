### Инструкция по использованию скрипта
* Скачать репозиторий с Github
  * git clone https://github.com/KrimUp/End.git
* Сделать файл **end.sh** испольнительным
  * chmod +x end.sh
* Запустить скрипт и следить за установкой :)
  * ./end.sh
